# Falcon Academy 2nd Grade Learning Portal

An interactive learning website designed for 2nd grade students at Falcon Academy. The site combines reading, math, missions, rewards, and progress tracking into a game-style learning experience that students can use independently.

---

## Purpose

The goal of this site is to:

- Build reading and math skills through short missions
- Encourage independent practice
- Reward effort and persistence
- Create a positive Falcon-themed learning experience
- Allow students to continue where they left off

---

## Student Features

### Reading Missions
Students complete reading passages and answer comprehension questions.

Features:
- 1st Grade Mode
- 2nd Grade Mode
- Progress saving
- Resume unfinished missions
- Instant feedback
- Mission completion rewards

### Math Missions
Students solve math problems aligned to grade-level standards.

Features:
- Skill practice
- Mission tracking
- Progress saving
- Reward system

### Falcon Rewards
Students earn rewards for completing missions.

Examples:
- Badges
- Points
- Falcon achievements
- Special unlocks

### Progress Tracking
Student progress is automatically saved in the browser.

Progress includes:
- Completed missions
- Scores
- Reading progress
- Reward status
- Unlocks

---

## How Progress Saving Works

Progress is stored locally on the device using browser storage.

Important:

- Progress stays on the same device/browser.
- Clearing browser data may erase progress.
- Progress does not currently sync between devices.

1. Short tasks
2. Immediate success opportunities
3. Visible progress
4. Student choice
5. Positive reinforcement

The goal is to build confidence while practicing essential skills.
